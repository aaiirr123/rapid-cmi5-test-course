# What Is Rapid CMI5?

**Rapid CMI5** is an open-source authoring and packaging tool for creating **cmi5-compliant training courses**, allowing instructional designers and developers to build, structure, and export learning content that integrates with **LMS platforms and Learning Record Stores (LRS)** using the **cmi5/xAPI standard**.

In practice, it enables teams to:

* **Author structured training content** (courses, blocks, and assignable units)
* **Package courses as cmi5 content packages**
  * **All the content is self contained and available in an air gapped environment**
* **Launch training experiences that report learning activity via xAPI**
* **Integrate with LMS platforms and cyber-range training environments**
