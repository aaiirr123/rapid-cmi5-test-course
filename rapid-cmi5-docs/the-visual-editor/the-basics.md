# The Toolbar

The **Toolbar** allows you to quickly edit and build your **CMI5 course content**.

<img alt="Toolbar Overview" id="20260305101219-a86df166-851a-4b3e-88b1-961077a6aad1" src="./Assets/Images/Screenshot 2026-03-05 101205.png" />

We designed the toolbar to feel similar to a **traditional document editor** so that most users can start creating content immediately without a steep learning curve.

The options on the **top-left side of the toolbar** should look very familiar.
These are the **basic text formatting tools** commonly found in word processors and document editors.

They allow you to quickly add:

* headings
* lists
* formatted text
* links
* basic document structure

***

# Media Tools

Towards the **middle of the toolbar**, you will find the **Media tools**.

<img width="201" height="51" alt="Media Tools" id="20260305101451-36255292-96f6-43b5-9da1-d73f6fb921dd" src="./Assets/Images/Screenshot 2026-03-05 101439.png" />

These allow you to easily insert rich media into your lesson content.

Supported media types include:

* **Images** – Add diagrams, screenshots, or illustrations
* **Audio** – Embed narration or recorded explanations
* **Video** – Include demonstrations or walkthroughs
* **File Downloads** – Provide lab files or additional resources
* **Code Blocks** – Display formatted code examples

Media elements help make lessons more **engaging and interactive for learners**.

***

# Layout Tools

The toolbar also includes several **layout features** that allow you to structure your content in visually appealing ways.

<img width="332" height="45" alt="Layout Tools" id="20260305101729-1959cdac-ebf9-404d-9862-fe37f01a5941" src="./Assets/Images/Screenshot 2026-03-05 101718.png" />

These tools allow you to organize content using modern layouts.

### Available Layout Options

**Grids**
Creates columns so content can appear side-by-side.

**Layout Box**
Allows control over alignment, spacing, and justification of content.

**Accordion**
Allows sections of content to be collapsed and expanded by the learner.

**Stepper**
Displays content in a step-by-step interactive format.

**Table**
Displays structured data in a table format.

**Tabs**
Organizes content into selectable tabs so users can switch between sections.

**Thematic Break**
Adds a horizontal divider to visually separate sections of content.

These layout options help authors create **clear, organized, and visually engaging lessons**.

## Lesson Appearance

The **Lesson Appearance** button allows you to make changes that affect the entire lesson.

Examples include:

* adjusting the **content width**
* changing **text alignment**
* modifying **padding and spacing**

These settings help ensure your lesson maintains a **consistent visual style**.

***

## Animation Library

The **Animation Library** allows authors to create animated content elements.

Animations can be used to:

* introduce key concepts
* highlight important information
* guide learners through complex material

These animated elements appear when a slide or section becomes visible, helping improve **learner engagement and focus**.

The animation tools are explained in more detail in the following slides.

## Block Library

The **Block Library** allows authors to quickly insert pre-built content components.

These include:

* **Activities**
* **Admonitions (notes, warnings, tips)**
* **Layout components**
* **Media elements**

Using the block library can significantly **speed up lesson development** by providing reusable building blocks.
