# Basic Markdown

Rapid Cmi5 is a markdown editor first and foremost, all of the normal markdown tricks still apply here.

***

# 1. Headings

Headings are one of the primary ways you can achieve a visual hierarchy in markdown&#x20;

# Heading 1

## Heading 2

### Heading 3

#### Heading 4

##### Heading 5

###### Heading 6

***

# 2. Text Formatting

**Bold text**

*Italic text*

***Bold + Italic***

~~Strikethrough~~

`Inline code`

> Blockquote text

# 3. Lists

## Unordered List

* Item one
* Item two
  * Nested item
  * Nested item
* Item three

## Ordered List

1. First
2. Second
3. Third

## Task List

* [x] Completed task
* [ ] Incomplete task
* [ ] Another task
