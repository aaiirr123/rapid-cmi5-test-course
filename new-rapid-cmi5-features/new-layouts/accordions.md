:::accordionContent{title="Example: FAQ Answer"}
**Question:** When should I use an accordion instead of separate slides?

Use an accordion when each section is short, optional, and related to the same topic. If the content must be read in order, use steps. If each section needs a full explanation or activity, split it into separate slides.
:::

:::accordionContent{title="Example: Key Terms"}
**AU:** Assignable Unit, usually a lesson in a CMI5 course.

**MoveOn:** The completion rule that tells the LMS when the learner has satisfied the AU requirement.

**xAPI statement:** A tracking record that describes what the learner did, such as launched, completed, passed, or answered.
:::

:::accordionContent{title="Example: Optional Deep Dive"}
Accordions work well for extra context that some learners may want but everyone does not need immediately.

For example, a compliance lesson might show the required rule on the slide and place policy background, exceptions, and examples in expandable sections.
